# Launch Audit

Run a bounded Sophia launch audit.

Scope:

- public page quality,
- no AI slop,
- no fake live claims,
- no old cartoon/play-button branding,
- no stale country/category claims,
- no mobile overlap,
- no horizontal overflow,
- no console errors,
- primary CTAs visible and useful.

Use project commands where appropriate:

```bash
npm run check
npm run audit:playwright
```

Return:

```text
Objective:
Pages checked:
Findings:
Verification:
Blockers:
Next action:
Approval needed:
```

Do not deploy. Do not touch secrets, billing, account settings, publishing, or paid tools.
