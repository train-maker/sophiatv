# Research Best

Use when Sophia needs the best way to build a feature, visual pattern, workflow, or agent setup.

Research order:

1. Official docs.
2. Existing Sophia repo/vault docs.
3. Trusted open-source examples.
4. Public design references as inspiration only.

For repos:

- verify official repo,
- check license,
- check maintenance,
- inspect install scripts before recommending,
- do not copy code blindly,
- keep paid/account/API-key needs behind approval.

Save durable findings to:

```text
.claude/memory/research-log.md
```

Return:

```text
Sources checked:
Best option:
Why:
Risks:
Sophia implementation path:
Approval needed:
```
