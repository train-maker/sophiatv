# Sophia Design System Memory

## System Name

SOVEREIGN.

## Visual North Star

Black diamond globe, gold cartography, real materials, jewel light, market-floor gravity, restrained luxury, and practical global access.

## Approved Assets

```text
assets/brand/sophia-logo-premium.png
```

## Hard Bans

- no cartoon/play-button Sophia branding,
- no generic AI blobs/orbs/mesh backgrounds,
- no purple-on-black neon sludge,
- no fake live counters or fake activity,
- no emoji UI on serious launch surfaces,
- no unsupported superlatives,
- no church/biblical framing,
- no template-card clutter,
- no mobile overlap or horizontal overflow.

## Required Feel

The page should feel like a real global business floor, not a demo. Status comes from structure: country mosaic, anchor seats, application lanes, translated inquiry paths, and honest proof.

## Design Checks

- one clear primary action,
- readable mobile hierarchy,
- stable dimensions for visual modules,
- no nested card clutter,
- real product/proof signal above decorative atmosphere,
- motion only when it clarifies or elevates,
- browser verification before claiming done.
