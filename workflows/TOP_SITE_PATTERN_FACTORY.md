# Top Site Pattern Factory

## Rule

Study expert public products, extract visible product patterns, then build original Sophia implementations. Do not copy proprietary source, assets, text, brand identity, logged-in surfaces, or private APIs.

## Clean Operating Language

Use "public product teardown," "visible UX," "interaction pattern," "motion/media strategy," "conversion flow," "public stack signal," and "original Sophia implementation."

## Seed Pattern Board

| Source | Public pattern to study | Sophia adaptation |
|---|---|---|
| YouTube | Media controls, captions, transcript, mini-player, creator workflow | Sophia player with translation, transcript, chapters, and creator publish checklist |
| Apple | Cinematic product object, restrained motion, hero clarity | 4K-style Earth hero with clean composition and fewer dashboard effects |
| Linear | Sequential workflow, keyboard clarity, calm premium UI | Sophia command surfaces that feel fast and organized |
| Netflix | Preview-first browsing and watch-state cues | Marketplace/media previews that help users choose without clutter |
| Stripe | Trust-building copy and technical polish | Clear pricing, payment confidence, and developer-grade simplicity |
| Shopify | Merchant onboarding and conversion steps | Better business listing submission and owner workflow |
