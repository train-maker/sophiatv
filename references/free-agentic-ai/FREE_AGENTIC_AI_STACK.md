# Sophia Free Agentic AI Stack

Purpose: use every useful free/no-spend AI tool without risking money, secrets, accounts, or production access.

## Rule

No-spend first. A tool is not automatically approved just because it has a free tier.

Approved without a new gate:

- local Ollama models already installed,
- local scripts and browser audits,
- local repo/document review,
- Claude Code inside the existing plan when no secrets, billing, publishing, or account permissions are involved,
- Cursor local project review when no account changes or paid extensions are required.

Still requires Tarence approval:

- API keys,
- billing setup,
- paid credits,
- account permission changes,
- public deployment/publishing,
- sending private data, secrets, customer data, account pages, or sensitive business records to cloud models.

## Source Notes

- Ollama runs local models and is the default no-spend path.
- Continue and Cline both document Ollama/local-model usage.
- Open WebUI documents Ollama connection and local model management.
- OpenRouter documents a free model router, but it still requires an API key and external prompt routing.
- Gemini API has a published free tier, but it still requires API key/project handling and rate-limit management.
- Groq documents a Free tier and a paid Developer tier; any upgrade remains blocked.
